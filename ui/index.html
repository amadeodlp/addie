What's an API key? It's a password that lets Addie talk to a language model — the AI that reads your session and responds to your messages. Without it, Addie is just an empty shell.

What's a language model? Think of it as the brain behind Addie. It reads your track names, devices, and parameters, understands what you're asking, and decides what to do. Without one, Addie can't do anything useful.

Which model do I need? You need one with at least 70 billion parameters — "70B" is a rough measure of how capable a model is. Addie sends your entire session in every message, which is a lot of information. Smaller models lose track of it and start making mistakes. 70B is the practical floor for reliable results.

Good options: Anthropic (Claude, best quality), DeepSeek (very capable, very cheap), OpenAI (ChatGPT), or OpenRouter (one key for all providers).

What's an API key? Addie needs to talk to an AI language model — a program that understands language and can reason about your session. Think of the key as a password that gives Addie access to that AI. You get it from the provider's website, paste it here once, and you're done.

What's a language model? It's the brain behind Addie. It reads your track names, devices, and parameters, understands what you're asking, and decides what to do. Without one, Addie is just a UI with no intelligence.

Which model do I need? You need one with at least 70 billion parameters — "70B" is a rough measure of how capable a model is. Good options:

  • Anthropic (Claude) — the best results overall, especially for nuanced mix advice. console.anthropic.com
